# Налаштування Formspree для форм

## Що таке Formspree?

Formspree - це найпростіший спосіб відправки форм без backend. Він безкоштовний для до 50 відправок на місяць.

## Крок 1: Створення форм на Formspree

1. Перейдіть на [formspree.io](https://formspree.io)
2. Зареєструйтеся або увійдіть
3. Створіть **дві нові форми**:
   - **Форма 1**: "Contact Form" (для секції консультації)
   - **Форма 2**: "Partnership Form" (для секції партнерства)

## Крок 2: Отримання endpoint URL

Після створення кожної форми, ви отримаєте endpoint URL, наприклад:
- `https://formspree.io/f/xjvqkzpn` (для контактної форми)
- `https://formspree.io/f/mnopqrst` (для форми партнерства)

## Крок 3: Налаштування змінних середовища

Створіть файл `.env.local` в корені проєкту (якщо його ще немає) та додайте:

```env
NEXT_PUBLIC_FORMSPREE_CONTACT_ENDPOINT=https://formspree.io/f/YOUR_CONTACT_FORM_ID
NEXT_PUBLIC_FORMSPREE_PARTNERSHIP_ENDPOINT=https://formspree.io/f/YOUR_PARTNERSHIP_FORM_ID
```

**Замініть:**
- `YOUR_CONTACT_FORM_ID` на ID вашої контактної форми
- `YOUR_PARTNERSHIP_FORM_ID` на ID вашої форми партнерства

## Крок 4: Перезапуск сервера

Після додавання змінних середовища, перезапустіть сервер розробки:

```bash
npm run dev
```

## Крок 5: Налаштування email на Formspree

1. Перейдіть до налаштувань кожної форми на Formspree
2. Додайте email, на який будуть приходити заявки
3. Перевірте email (Formspree надішле лист для підтвердження)

## Перевірка роботи

1. Заповніть форму на сайті
2. Натисніть "ВІДПРАВИТИ ЗАЯВКУ"
3. Перевірте email - має прийти лист з даними форми

## Важливо!

- Без налаштування змінних середовища форми будуть використовувати placeholder URL
- Formspree безкоштовний план: до 50 відправок на місяць
- Для продакшену рекомендовано налаштувати змінні середовища на хостингу

## Альтернативи

Якщо Formspree не підходить, можна використати:
- **EmailJS** - потрібно встановити пакет `@emailjs/browser`
- **Власний API** - створити Next.js API route
- **SendGrid/Mailgun** - для великих обсягів
